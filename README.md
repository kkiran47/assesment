# CreatorDiscover 🌟

CreatorDiscover is a premium, modern SaaS dashboard designed for finding, analyzing, and connecting with top creators and influencers across all major platforms. Built with a focus on world-class aesthetics and seamless user experience.

## ✨ Features

- **Premium UI/UX:** Glassmorphism, dynamic gradients, and fluid physics-based animations.
- **Advanced Filtering:** Filter creators by platform, category, and price range.
- **Global State Management:** Lightning-fast state updates powered by Zustand.
- **Interactive Animations:** Interactive cursor-following glow, staggered grid loading, and 3D card hover effects.
- **Dark Mode Support:** First-class dark mode tailored for a professional dashboard experience.
- **Responsive Design:** Mobile-first architecture ensuring perfect layouts across all screen sizes.

## 🛠️ Tech Stack

- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS (v4)
- **Animations:** Framer Motion
- **Icons:** React Icons
- **State Management:** Zustand

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js (v18+) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kkiran47/assesment.git
   ```

2. Navigate into the project directory (or use your local folder name):
   ```bash
   cd assesment
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit the local URL provided by Vite (usually `http://localhost:5173`).
